Функция ArraySize

Описание

Функция ArraySize возвращает количество элементов в массиве.

Параметры

Параметр Тип Обязательный Описание
value Массив Да Массив, размер которого нужно получить

Возвращаемое значение

Возвращает число, представляющее количество элементов в массиве.

Примеры

Базовое использование

{
  "function": "ArraySize",
  "value": ["яблоко", "банан", "вишня"]
}

Результат: 3

С результатом функции

{
  "function": "ArraySize",
  "value": {
    "function": "CombineArrays",
    "arrays": [
      [1, 2, 3],
      [4, 5, 6]
    ]
  }
}

Результат: 6

Пустой массив

{
  "function": "ArraySize",
  "value": []
}

Результат: 0

Условная логика

{
  "function": "Equals",
  "value1": {
    "function": "ArraySize",
    "value": ["элемент1", "элемент2", "элемент3"]
  },
  "value2": 3,
  "trueResult": "Массив содержит ровно 3 элемента",
  "falseResult": "Размер массива не равен 3"
}

Результат: "Массив содержит ровно 3 элемента"

Обработка ошибок

Если параметр value не является массивом, функция выбросит ошибку:

{
  "function": "ArraySize",
  "value": "не массив"
}

Ошибка: ArraySize function: value parameter must be an array, got string

Случаи использования

  • Проверка, пуст ли массив или содержит определенное количество элементов
  • Условная логика на основе размера массива
  • Валидация и обработка ошибок
  • Динамическая генерация UI на основе размера данных
