Функция DateFormat

Описание

Функция DateFormat форматирует строку даты согласно указанному шаблону форматирования.

Параметры

Параметр Тип Обязательный Описание
date Строка Да Строка даты для форматирования (рекомендуется ISO 8601 формат)
format Строка Да Шаблон форматирования для применения

Шаблоны форматирования

Шаблон Описание Пример
YYYY 4-значный год 2025
YY 2-значный год 25
MM 2-значный месяц 01-12
M Месяц без ведущего нуля 1-12
DD 2-значный день 01-31
D День без ведущего нуля 1-31
HH 2-значный час (24-часовой) 00-23
H Час без ведущего нуля (24-часовой) 0-23
mm 2-значная минута 00-59
m Минута без ведущего нуля 0-59
ss 2-значная секунда 00-59
s Секунда без ведущего нуля 0-59
SSS 3-значные миллисекунды 000-999

Возвращаемое значение

Возвращает отформатированную строку даты согласно указанному шаблону форматирования.

Примеры

Базовое использование

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "YYYY-MM-DD HH:mm:ss"
}

Результат: "2025-10-22 15:30:45"

Российский формат даты

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "DD.MM.YYYY"
}

Результат: "22.10.2025"

Только время

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "HH:mm"
}

Результат: "15:30"

С параметрами функций

{
  "function": "DateFormat",
  "date": "{{response.body.createdAt}}",
  "format": "YYYY-MM-DD"
}

Результат: "2025-10-22" (если createdAt равен "2025-10-22T15:30:45.123Z")

Сложный формат

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "DD/MM/YY HH:mm:ss.SSS"
}

Результат: "22/10/25 15:30:45.123"

Обработка ошибок

Если строка даты недействительна, функция выбросит ошибку:

{
  "function": "DateFormat",
  "date": "недействительная-дата",
  "format": "YYYY-MM-DD"
}

Ошибка: DateFormat function: invalid date format "недействительная-дата"

Случаи использования

  • Форматирование временных меток для отображения
  • Преобразование дат между различными форматами
  • Создание удобных для пользователя отображений дат
  • Генерация отформатированных дат для сообщений и уведомлений
