IsNotEmpty

Функция IsNotEmpty проверяет, что значение не является пустой строкой, null или undefined.

Синтаксис

{
  "function": "IsNotEmpty",
  "value": "значение для проверки"
}

Параметры

Параметр Тип Обязательный Описание
value any Да Значение для проверки (может быть интерполируемым полем)

Возвращаемое значение

  • true - если значение не пустое, не null и не undefined
  • false - если значение является пустой строкой, null или undefined

Примеры использования

Пример 1: Проверка строки

{
  "function": "IsNotEmpty",
  "value": "Hello World"
}
// Возвращает: true

Пример 2: Проверка пустой строки

{
  "function": "IsNotEmpty",
  "value": ""
}
// Возвращает: false

Пример 3: Проверка null

{
  "function": "IsNotEmpty",
  "value": null
}
// Возвращает: false

Пример 4: Проверка интерполируемого поля

{
  "function": "IsNotEmpty",
  "value": "{{response.body.phone}}"
}
// Возвращает: true если поле существует и не пустое, false в противном случае

Пример 5: Проверка строки с пробелами

{
  "function": "IsNotEmpty",
  "value": "   "
}
// Возвращает: false (пробелы считаются пустой строкой)

Примечания

  • Функция автоматически обрезает пробелы в начале и конце строки перед проверкой
  • Числа, объекты и массивы считаются непустыми значениями
  • Полезно для проверки наличия данных перед их использованием
  • Может использоваться в условиях для условной логики
