Функция Compare

Описание

Функция Compare сравнивает два числовых значения с использованием указанного оператора и возвращает логический результат.

Параметры

Параметр Тип Обязательный Описание
value1 Число Да Первое значение для сравнения
value2 Число Да Второе значение для сравнения
operator Строка Да Оператор сравнения для использования

Поддерживаемые операторы

Оператор Альтернатива Описание Пример
more > Больше чем 5 > 3 = true
moreThanOrEquals >= Больше или равно 5 >= 5 = true
equals == или === Равно 5 == 5 = true
less < Меньше чем 3 < 5 = true
lessThanOrEquals <= Меньше или равно 3 <= 5 = true

Возвращаемое значение

Возвращает true или false на основе результата сравнения.

Примеры

Базовое использование

{
  "function": "Compare",
  "value1": 10,
  "value2": 5,
  "operator": "more"
}

Результат: true

Сравнение на равенство

{
  "function": "Compare",
  "value1": 10,
  "value2": 10,
  "operator": "equals"
}

Результат: true

Сравнение "меньше чем"

{
  "function": "Compare",
  "value1": 3,
  "value2": 7,
  "operator": "less"
}

Результат: true

С параметрами функций

{
  "function": "Compare",
  "value1": "{{response.body.balance}}",
  "value2": 1000,
  "operator": "moreThanOrEquals"
}

Результат: true (если balance равен 1500)

Альтернативный синтаксис оператора

{
  "function": "Compare",
  "value1": 5,
  "value2": 3,
  "operator": ">"
}

Результат: true

Условная логика

{
  "function": "Switch",
  "value": {
    "function": "Compare",
    "value1": "{{response.body.score}}",
    "value2": 80,
    "operator": "more"
  },
  "cases": [
    {
      "match": true,
      "result": "Отличный результат!"
    },
    {
      "match": false,
      "result": "Хороший результат, но можно улучшить"
    }
  ]
}

Результат: "Отличный результат!" (если score > 80)

Обработка ошибок

Если какой-либо параметр не является действительным числом, функция выбросит ошибку:

{
  "function": "Compare",
  "value1": "не число",
  "value2": 5,
  "operator": "more"
}

Ошибка: Compare function: value1 "не число" is not a valid number

Если используется неподдерживаемый оператор:

{
  "function": "Compare",
  "value1": 5,
  "value2": 3,
  "operator": "неверный"
}

Ошибка: Compare function: unsupported operator "неверный"

Случаи использования

  • Валидация числовых условий
  • Реализация бизнес-логики
  • Создание условных рабочих процессов
  • Сравнение пользовательского ввода с пороговыми значениями
  • Построение систем оценки и рейтинга
