üîß Value processing

Value types

Function processes different types of values in array:

  • Strings: Interpolated with variables
  • Functions: Executed and result converted to string
  • Numbers: Converted to string
  • null/undefined: Ignored

Empty value filtering

Automatically filtered:

  • Empty strings ("")
  • null values
  • undefined values
  • Function results returning empty strings
