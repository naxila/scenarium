DateFormat Function

Description

The DateFormat function formats a date string according to a specified format pattern.

Parameters

Parameter Type Required Description
date String Yes The date string to format (ISO 8601 format recommended)
format String Yes The format pattern to apply

Format Patterns

Pattern Description Example
YYYY 4-digit year 2025
YY 2-digit year 25
MM 2-digit month 01-12
M Month without leading zero 1-12
DD 2-digit day 01-31
D Day without leading zero 1-31
HH 2-digit hour (24-hour) 00-23
H Hour without leading zero (24-hour) 0-23
mm 2-digit minute 00-59
m Minute without leading zero 0-59
ss 2-digit second 00-59
s Second without leading zero 0-59
SSS 3-digit milliseconds 000-999

Return Value

Returns a formatted date string according to the specified format pattern.

Examples

Basic Usage

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "YYYY-MM-DD HH:mm:ss"
}

Result: "2025-10-22 15:30:45"

Russian Date Format

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "DD.MM.YYYY"
}

Result: "22.10.2025"

Time Only

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "HH:mm"
}

Result: "15:30"

With Function Parameters

{
  "function": "DateFormat",
  "date": "{{response.body.createdAt}}",
  "format": "YYYY-MM-DD"
}

Result: "2025-10-22" (if createdAt is "2025-10-22T15:30:45.123Z")

Complex Format

{
  "function": "DateFormat",
  "date": "2025-10-22T15:30:45.123Z",
  "format": "DD/MM/YY HH:mm:ss.SSS"
}

Result: "22/10/25 15:30:45.123"

Error Handling

If the date string is invalid, the function will throw an error:

{
  "function": "DateFormat",
  "date": "invalid-date",
  "format": "YYYY-MM-DD"
}

Error: DateFormat function: invalid date format "invalid-date"

Use Cases

  • Formatting timestamps for display
  • Converting dates between different formats
  • Creating user-friendly date displays
  • Generating formatted dates for messages and notifications
