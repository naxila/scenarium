üìã Description

Equals is a function for comparing an array of values with conditional result return. Supports variable interpolation and nested functions in results.

üéØ Purpose

  • Conditional logic in scenarios

üîß How it works

Compare values

Function compares all values in the values array:

  • All values are interpolated before comparison
  • Comparison is strict (===)
  • If all values are equal - returns trueResult
  • If at least one value differs - returns falseResult

Result processing

Results trueResult and falseResult:

  • Can be any data types
  • Support variable interpolation
  • Can contain nested functions
  • If result not specified - returns true or false

‚ö†Ô∏è Important notes

  • Performance: All values are interpolated before comparison
