IsNotEmpty

The IsNotEmpty function checks that a value is not an empty string, null, or undefined.

Syntax

{
  "function": "IsNotEmpty",
  "value": "value to check"
}

Parameters

Parameter Type Required Description
value any Yes Value to check (can be an interpolated field)

Return Value

  • true - if the value is not empty, not null, and not undefined
  • false - if the value is an empty string, null, or undefined

Usage Examples

Example 1: Checking a string

{
  "function": "IsNotEmpty",
  "value": "Hello World"
}
// Returns: true

Example 2: Checking an empty string

{
  "function": "IsNotEmpty",
  "value": ""
}
// Returns: false

Example 3: Checking null

{
  "function": "IsNotEmpty",
  "value": null
}
// Returns: false

Example 4: Checking an interpolated field

{
  "function": "IsNotEmpty",
  "value": "{{response.body.phone}}"
}
// Returns: true if the field exists and is not empty, false otherwise

Example 5: Checking a string with spaces

{
  "function": "IsNotEmpty",
  "value": "   "
}
// Returns: false (spaces are considered an empty string)

Notes

  • The function automatically trims whitespace from the beginning and end of strings before checking
  • Numbers, objects, and arrays are considered non-empty values
  • Useful for checking data availability before using it
  • Can be used in conditions for conditional logic
