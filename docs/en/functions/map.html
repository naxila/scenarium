Dynamic button generation function that creates inline actions from array data with variable interpolation support.

üìã Description

Map is a special function for creating dynamic inline actions (buttons) from array data. It generates buttons for each element in an array, allowing users to interact with data dynamically.

üéØ Purpose

  • Creating dynamic buttons from arrays
  • Generating interactive menus
  • Building data-driven interfaces
  • Creating user selection options

Parameter Type Required Description
items array ‚úÖ Array of data to create buttons from
forEach object ‚úÖ Template for each button with title and onClick action

üí° Usage Examples

Creating Buttons from Array

{
  "action": "SendMessage",
  "text": "Choose a color:",
  "inlineActions": {
    "function": "Map",
    "items": "{{colors}}",
    "forEach": {
      "title": "{{it}}",
      "onClick": {
        "action": "SendMessage",
        "text": "You selected: {{it}}"
      }
    }
  }
}

Result: Creates buttons for each color in the array

Buttons with Index

    }

}

Result: Creates numbered buttons for each product

Generating Text from Array

üîó Available Variables

In forEach Template

  • {{forEach}} - Template object for each element

‚ö†Ô∏è Important Notes

  • Items required: Must provide a valid array in the items parameter
  • forEach required: Must specify a template object for each element
  • Item variable: Each element is available as {{it}} in the forEach template
  • Index variable: Element index is available as {{index}} in the forEach template
  • Interpolation: All string values are interpolated before processing
  • Button creation: When used in inlineActions, creates interactive buttons
  • Text generation: When used in text, generates formatted text from array
