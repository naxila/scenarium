ArraySize Function

Description

The ArraySize function returns the number of elements in an array.

Parameters

Parameter Type Required Description
value Array Yes The array whose size you want to get

Return Value

Returns a number representing the count of elements in the array.

Examples

Basic Usage

{
  "function": "ArraySize",
  "value": ["apple", "banana", "cherry"]
}

Result: 3

With Function Result

{
  "function": "ArraySize",
  "value": {
    "function": "CombineArrays",
    "arrays": [
      [1, 2, 3],
      [4, 5, 6]
    ]
  }
}

Result: 6

Empty Array

{
  "function": "ArraySize",
  "value": []
}

Result: 0

Conditional Logic

{
  "function": "Equals",
  "value1": {
    "function": "ArraySize",
    "value": ["item1", "item2", "item3"]
  },
  "value2": 3,
  "trueResult": "Array has exactly 3 items",
  "falseResult": "Array size is not 3"
}

Result: "Array has exactly 3 items"

Error Handling

If the value parameter is not an array, the function will throw an error:

{
  "function": "ArraySize",
  "value": "not an array"
}

Error: ArraySize function: value parameter must be an array, got string

Use Cases

  • Checking if an array is empty or has a specific number of elements
  • Conditional logic based on array size
  • Validation and error handling
  • Dynamic UI generation based on data size
