Compare Function

Description

The Compare function compares two numeric values using a specified operator and returns a boolean result.

Parameters

Parameter Type Required Description
value1 Number Yes The first value to compare
value2 Number Yes The second value to compare
operator String Yes The comparison operator to use

Supported Operators

Operator Alternative Description Example
more > Greater than 5 > 3 = true
moreThanOrEquals >= Greater than or equal 5 >= 5 = true
equals == or === Equal to 5 == 5 = true
less < Less than 3 < 5 = true
lessThanOrEquals <= Less than or equal 3 <= 5 = true

Return Value

Returns true or false based on the comparison result.

Examples

Basic Usage

{
  "function": "Compare",
  "value1": 10,
  "value2": 5,
  "operator": "more"
}

Result: true

Equal Comparison

{
  "function": "Compare",
  "value1": 10,
  "value2": 10,
  "operator": "equals"
}

Result: true

Less Than Comparison

{
  "function": "Compare",
  "value1": 3,
  "value2": 7,
  "operator": "less"
}

Result: true

With Function Parameters

{
  "function": "Compare",
  "value1": "{{response.body.balance}}",
  "value2": 1000,
  "operator": "moreThanOrEquals"
}

Result: true (if balance is 1500)

Alternative Operator Syntax

{
  "function": "Compare",
  "value1": 5,
  "value2": 3,
  "operator": ">"
}

Result: true

Conditional Logic

{
  "function": "Switch",
  "value": {
    "function": "Compare",
    "value1": "{{response.body.score}}",
    "value2": 80,
    "operator": "more"
  },
  "cases": [
    {
      "match": true,
      "result": "Excellent score!"
    },
    {
      "match": false,
      "result": "Good score, but can be improved"
    }
  ]
}

Result: "Excellent score!" (if score > 80)

Error Handling

If any parameter is not a valid number, the function will throw an error:

{
  "function": "Compare",
  "value1": "not a number",
  "value2": 5,
  "operator": "more"
}

Error: Compare function: value1 "not a number" is not a valid number

If an unsupported operator is used:

{
  "function": "Compare",
  "value1": 5,
  "value2": 3,
  "operator": "invalid"
}

Error: Compare function: unsupported operator "invalid"

Use Cases

  • Validating numeric conditions
  • Implementing business logic rules
  • Creating conditional workflows
  • Comparing user input with thresholds
  • Building scoring and rating systems
