What is a Scenario?

A scenario is a JSON file that describes the behavior of your Telegram bot. It defines menus, actions, and the flow of user interaction.

Scenario Structure

A scenario consists of several main sections:

Basic Structure

{
  "onStartActions": [
    {
      "action": "Navigate",
      "menuItem": "Main"
    }
  ],
  "menuItems": {
    "Main": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Welcome to the bot!"
      }
    }
  }
}

Main Sections

onStartActions

Actions that are executed when the bot starts or when a user sends /start command.

Start Parameters (startParams)

Scenarium automatically parses parameters from the /start command and makes them available in onStartActions through the startParams object:

Supported formats:
  • Telegram links: ?start=123 â†’ {{startParams.start}}
  • URL parameters: param1=value1¶m2=value2
  • Space-separated parameters: name John age 25
Available variables in onStartActions:
{
  "onStartActions": [
    {
      "action": "SendMessage",
      "text": "Welcome! Parameters: {{startParams.start}}\nUser: {{telegram.firstName}}"
    }
  ]
}

Telegram object

In onStartActions, the telegram object is available with user and chat information:

  • {{telegram.chatId}} - Chat ID
  • {{telegram.userId}} - User ID
  • {{telegram.firstName}} - User first name
  • {{telegram.username}} - User username
  • {{telegram.chatType}} - Chat type (private, group, etc.)

menuItems

Defines all available menus and their behavior. Each menu can have:

  • onNavigation - Actions when user enters the menu
  • onInput - Actions when user sends text input
  • onCallback - Actions when user clicks inline buttons

Example Scenarios

Simple Welcome Bot

{
  "onStartActions": [
    {
      "action": "Navigate",
      "menuItem": "Welcome"
    }
  ],
  "menuItems": {
    "Welcome": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Hello! Welcome to our bot!",
        "inlineActions": [
          {
            "title": "Get Help",
            "onClick": {
              "action": "Navigate",
              "menuItem": "Help"
            }
          }
        ]
      }
    },
    "Help": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "This is the help section. How can I assist you?"
      }
    }
  }
}
