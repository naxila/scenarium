What is a Scenario?

A scenario is a JSON file that describes the behavior of your Telegram bot. It defines menus, actions, and the flow of user interaction.

Scenario Structure

A scenario consists of several main sections:

Basic Structure

{
  "onStartActions": [
    {
      "action": "Navigate",
      "menuItem": "Main"
    }
  ],
  "menuItems": {
    "Main": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Welcome to the bot!"
      }
    }
  }
}

Main Sections

onStartActions

Actions that are executed when the bot starts or when a user sends /start command.

menuItems

Defines all available menus and their behavior. Each menu can have:

  • onNavigation - Actions when user enters the menu
  • onInput - Actions when user sends text input
  • onCallback - Actions when user clicks inline buttons

Example Scenario

Simple Welcome Bot

{
  "onStartActions": [
    {
      "action": "Navigate",
      "menuItem": "Welcome"
    }
  ],
  "menuItems": {
    "Welcome": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Hello! Welcome to our bot!",
        "inlineActions": [
          {
            "title": "Get Help",
            "onClick": {
              "action": "Navigate",
              "menuItem": "Help"
            }
          }
        ]
      }
    },
    "Help": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "This is the help section. How can I assist you?"
      }
    }
  }
}
