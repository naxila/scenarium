Learn how to use variable interpolation and functions in your bot messages.

What is Interpolation?

Interpolation allows you to insert dynamic values and execute functions in your message text using the {{variableName}} syntax.

Available Variables

userContext

Contains information about the current user:

  • {{userId}} - User's Telegram ID
  • {{name}} - User's name
  • {{currentMenu}} - Current menu name
  • {{lastMessageId}} - Last message ID

scenarioContext

Contains scenario configuration data:

  • {{apiUrl}} - API URL
  • {{version}} - Scenario version

localContext

Contains local execution data:

  • {{messageId}} - Current message ID
  • {{response}} - API response
  • {{error}} - Error message

input

Contains user input data:

  • {{userName}} - Entered name
  • {{email}} - Entered email

Using Functions

Functions are used in object syntax, not in variable interpolation:

Built-in Functions

{
  "action": "SendMessage",
  "text": {
    "function": "JoinToString",
    "values": [
      "Hello {{name}}! Today is {{date}}",
      "The sum of 5 and 3 is {{sum}}"
    ],
    "separator": " "
  }
}

Example Scenarios

Personalized Welcome

{
  "action": "SendMessage",
  "text": "Welcome, {{name}}! You are currently in the {{currentMenu}} menu."
}

Dynamic API Response

{
  "action": "SendMessage",
  "text": "API Response: {{response}}"
}

User Input Processing

{
  "action": "SendMessage",
  "text": "Thank you, {{userName}}! We received your email: {{email}}"
}

Nested Interpolation

You can use interpolation within function parameters:

Nested Example

{
  "action": "SendMessage",
  "text": "Hello {{name}}! The current time is {{Date:time}} and your ID is {{userId}}"
}

Error Handling

If a variable or function is not found, it will be replaced with an empty string. Make sure to handle missing data gracefully:

Safe Interpolation

{
  "action": "SendMessage",
  "text": "Hello {{name || 'User'}}! Welcome to our bot."
}
