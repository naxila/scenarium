What is Interpolation?

Interpolation allows you to insert dynamic values and execute functions in your message text using the {{variableName}} syntax.

Available Variables

startParams (Start Parameters)

Available only in onStartActions. Contains parameters from the /start command:

  • {{startParams.start}} - Parameter from Telegram link (?start=value)
  • {{startParams.param1}} - Any URL parameters (param1=value1)
  • {{startPayload}} - Raw parameter string

telegram (Telegram Information)

Available only in onStartActions. Contains user and chat information:

  • {{telegram.chatId}} - Chat ID
  • {{telegram.userId}} - User ID
  • {{telegram.firstName}} - User first name
  • {{telegram.lastName}} - User last name
  • {{telegram.username}} - User username
  • {{telegram.chatType}} - Chat type (private, group, etc.)
  • {{telegram.isBot}} - Is bot
  • {{telegram.languageCode}} - Language code
  • {{telegram.messageId}} - Message ID
  • {{telegram.date}} - Message date

userContext

Contains information about the current user:

  • {{currentMenu}} - Current menu name
  • {{lastMessageId}} - Last message ID

localContext

Contains local execution data:

  • {{messageId}} - Current message ID
  • {{response}} - API response
  • {{error}} - Error message

Using Functions

Functions are used in object syntax, not in variable interpolation:

Built-in Functions

{
  "action": "SendMessage",
  "text": {
    "function": "JoinToString",
    "values": [
      "Hello {{name}}! Today is {{date}}",
      "The sum of 5 and 3 is {{sum}}"
    ],
    "separator": " "
  }
}

Nested Interpolation

You can use interpolation within function parameters:

Nested Example

{
  "action": "SendMessage",
  "text": "Hello {{name}}! Your ID is {{userId}} and you are in {{currentMenu}}"
}

Error Handling

If a variable is not found, it will remain as the original variable syntax (e.g., {{unknownVariable}}). Make sure to handle missing data gracefully:

Safe Interpolation

{
  "action": "SendMessage",
  "text": "Hello {{name}}! Welcome to our bot."
}

Note about missing variables

If a variable is not found, it will be replaced with the original variable syntax (e.g., {{unknownVariable}}). Always ensure your variables are properly defined in the context.
