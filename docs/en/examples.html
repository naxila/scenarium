Basic Examples

Simple Welcome Bot

A basic bot that greets users and provides navigation.

{
  "onStartActions": [
    {
      "action": "Navigate",
      "menuItem": "Main"
    }
  ],
  "menuItems": {
    "Main": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Welcome to our bot! Choose an option:",
        "inlineActions": [
          {
            "title": "About",
            "onClick": {
              "action": "Navigate",
              "menuItem": "About"
            }
          },
          {
            "title": "Help",
            "onClick": {
              "action": "Navigate",
              "menuItem": "Help"
            }
          }
        ]
      }
    },
    "About": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "This is a Scenarium bot example."
      }
    },
    "Help": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Need help? Contact our support team."
      }
    }
  }
}

User Registration Bot

A bot that collects user information step by step.

{
  "onStartActions": [
    {
      "action": "Navigate",
      "menuItem": "Welcome"
    }
  ],
  "menuItems": {
    "Welcome": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Welcome! Let's get you registered.",
        "inlineActions": [
          {
            "title": "Start Registration",
            "onClick": {
              "action": "Navigate",
              "menuItem": "GetName"
            }
          }
        ]
      }
    },
    "GetName": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Please enter your name:"
      },
      "onInput": {
        "action": "Store",
        "key": "name",
        "value": "{{input.text}}",
        "onSuccess": {
          "action": "Navigate",
          "menuItem": "GetEmail"
        }
      }
    },
    "GetEmail": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Please enter your email:"
      },
      "onInput": {
        "action": "Store",
        "key": "email",
        "value": "{{input.text}}",
        "onSuccess": {
          "action": "Navigate",
          "menuItem": "Confirmation"
        }
      }
    },
    "Confirmation": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "Thank you, {{userContext.data.name}}! Your email is {{userContext.data.email}}"
      }
    }
  }
}

        },
        "onSuccess": {
          "action": "SendMessage",

Multi-language Bot

{
  "menuItems": {
    "Language": {
      "onNavigation": {
        "action": "SendMessage",
        "text": "{{Translate:hello:{{userContext.data.language || 'en'}}}}",
        "inlineActions": [
          {
            "title": "English",
            "onClick": {
              "action": "Store",
              "key": "language",
              "value": "en",
              "onSuccess": {
                "action": "Navigate",
                "menuItem": "Main"
              }
            }
          },
          {
            "title": "Russian",
            "onClick": {
              "action": "Store",
              "key": "language",
              "value": "ru",
              "onSuccess": {
                "action": "Navigate",
                "menuItem": "Main"

Best Practices

  • Use descriptive menu names
  • Handle errors gracefully
  • Provide clear navigation paths
  • Use interpolation for dynamic content
  • Test your scenarios thoroughly
